各ワールドへ入り方

メタ忍ワールドの各プラットフォームへの具体的な入り方を解説します。
スタンプラリーに参加するための第一歩をサポートします。

Sandbox（サンドボックス）のワールド

CNP Land への入り方

  1. Sandboxアプリを起動し、アカウントにログインします
  2. 検索バーで「CNP Land」と検索するか、公式リンクにアクセスします
  3. 「PLAY」ボタンをクリックしてワールドに入場します
  4. ロード完了後、WASD/矢印キーで移動、マウスで視点操作ができます
  5. 中央広場にあるフォトスポットを探して、スクリーンショットを撮影しましょう

ヒント：初回アクセス時はロード時間が長くなることがあります。また、PCスペックによってはグラフィック設定を下げると動作が安定します。

NOAランド への入り方

  1. Sandboxアプリを起動し、アカウントにログインします
  2. 検索バーで「NOA Land」と検索するか、公式リンクにアクセスします
  3. 「PLAY」ボタンをクリックしてワールドに入場します
  4. 忍者の里をモチーフにした広大なフィールドを探索しましょう
  5. 中央の大きな鳥居近くにフォトスポットがあります

クリプトニンジャ咲耶ワールド への入り方

  1. Sandboxアプリを起動し、アカウントにログインします
  2. 公式リンクからアクセスします
  3. 「PLAY」ボタンをクリックしてワールドに入場します
  4. アニメの世界観を再現した空間を探索しましょう
  5. 咲耶像の前がフォトスポットになっています

Rium（リウム）のワールド

クリプトニンジャ・オンライン への入り方

  1. ブラウザからRium公式サイトにアクセスします
  2. アカウント登録またはログインを行います
  3. 検索バーで「クリプトニンジャ・オンライン」を検索します
  4. ワールドカードをクリックして入場します
  5. 忍者アバターを操作して、中央広場のフォトスポットを探しましょう

ヒント：Riumはブラウザベースなので、Chrome/Edgeなどの最新ブラウザの使用を推奨します。

甲賀シティ への入り方

  1. ブラウザからRium公式サイトにアクセスします
  2. アカウント登録またはログインを行います
  3. 検索バーで「甲賀シティ」を検索します
  4. ワールドカードをクリックして入場します
  5. 忍者の里を再現した街並みを探索し、城の前にあるフォトスポットを見つけましょう

Cluster（クラスター）のワールド

ニンジャ寺子屋 への入り方

  1. PC、スマートフォン、またはVRデバイスでClusterアプリを起動します
  2. アカウントにログインします
  3. 検索機能で「ニンジャ寺子屋」を検索するか、直接リンクからアクセスします
  4. 入場後、伝統的な日本建築の寺子屋内を探索します
  5. 教室内の巻物の前がフォトスポットになっています

ヒント：Clusterではイベントが定期的に開催されています。イベント時間に合わせて訪問すると、他の参加者と交流できます。

Spatial（スペーシャル）のワールド

咲耶ポータル への入り方

  1. ブラウザから咲耶ポータルの直接リンクにアクセスします
  2. 「Enter Space」をクリックします
  3. アカウント登録またはログインを行います（Googleアカウントでの登録が簡単です）
  4. アバターを選択または作成します
  5. ポータル内のNFTギャラリーを探索し、中央展示エリアのフォトスポットを見つけましょう

ヒント：Spatialは高品質な3D表示を行うため、安定したインターネット接続と比較的高性能なデバイスを推奨します。

スクリーンショット撮影のコツ

  • 各ワールドのフォトスポットは、特別な装飾や看板で目立つようになっています
  • スクリーンショットはWindowsの場合「Win+PrintScreen」キー、Macの場合「Command+Shift+3」で撮影できます
  • 撮影したスクリーンショットは「#メタ忍ワールド」「#にんだおメタ旅」のハッシュタグを付けてXに投稿しましょう
  • 全7ワールドのスクリーンショットを撮影・投稿すると、CNGトークンがもらえます
