メタバースプラットフォームの始め方＋α

各メタバースプラットフォームの始め方を解説します。
初めての方でも安心してメタ忍ワールドに参加できるよう、詳しい手順をご紹介します。

Sandbox（サンドボックス）の始め方

CNP Land、NOAランド、クリプトニンジャ咲耶ワールドで利用

Sandboxはブロックチェーン技術を活用した3Dメタバースプラットフォームです。以下の手順で始めることができます：

  1. アカウント作成：公式サイトにアクセスし、メールアドレスで登録またはウォレット連携で登録
  2. アバター作成：自分だけのアバターをカスタマイズ
  3. クライアントのダウンロード：PCの場合はクライアントソフトをインストール
  4. ワールドへのアクセス：EXPERIENCESタブから目的のワールドを検索して入場

詳しい始め方は以下の公式ガイドをご覧ください：

Sandboxの詳細ガイドを見る →

Rium（リアム）の始め方

クリプトニンジャ・オンライン、甲賀シティで利用

Riumは日本発のメタバースプラットフォームで、ブラウザから簡単にアクセスできます：

  1. アカウント登録：公式サイトでメールアドレスまたはSNSアカウントで登録
  2. アバター設定：基本的なアバターをカスタマイズ
  3. ワールドへのアクセス：ブラウザから直接ワールドにアクセス可能
  4. 操作方法：WASDキーで移動、マウスで視点操作

詳しい始め方は以下のガイドをご覧ください：

Riumの詳細ガイドを見る →

Cluster（クラスター）の始め方

ニンジャ寺子屋で利用

Clusterは日本発のVRメタバースプラットフォームで、PCやスマートフォンから利用できます：

  1. アカウント作成：公式サイトまたはアプリでアカウントを作成
  2. アプリのダウンロード：PC版、スマホ版、VR版があります
  3. アバター設定：VRMファイルをアップロードするか、基本アバターを選択
  4. ワールドへのアクセス：ワールドURLまたはIDから直接アクセス

詳しい始め方は以下のガイドをご覧ください：

Clusterの詳細ガイドを見る →

Spatial（スペーシャル）の始め方

咲耶ポータルで利用

Spatialはブラウザベースのメタバースプラットフォームで、アートやNFTの展示に適しています：

  1. アカウント作成：公式サイトでGoogleアカウントまたはメールアドレスで登録
  2. ブラウザアクセス：専用アプリ不要でブラウザから直接アクセス可能
  3. アバター設定：基本アバターの選択またはカスタムアバターのアップロード
  4. ワールドへのアクセス：共有されたURLから直接スペースにアクセス
